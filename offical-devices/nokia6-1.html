NOKIA 6.1 (2018)

DEVICE NAME : NOKIA 6.1 (2018)
DEVICE CODE NAME : PL2 | PL2_sprout | Plate2
NOTES:-
      1) To install ancient os on your device your phones bootloader must be unlocked
      2) We are not responsible for your mistake
Maintainer :- ANKIT KENE
Device Offical Support Group :- TELEGRAM

Installation Instructions

Note - I don't recommend you people to flash any other custom kernels on this ROM
1. Boot in to twrp
2. Format data by typing yes
3. Reboot to bootloader
4. Now again boot in to twrp
5. Wipe everything
6. Now copy rom zip and vendor zip to phone
7. Flash the zips (Flash vendor zip on both slots)
8. Change to alternative slot
Example if you are on Slot a Change to slot b if you are on b change to a
9. Now wipe all except system & vendor
10. Reboot system
